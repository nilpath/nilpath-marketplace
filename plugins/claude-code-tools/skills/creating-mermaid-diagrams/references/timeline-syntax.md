# Timeline Syntax

Quick reference for Mermaid timelines.

## Basic Structure

```mermaid
timeline
    title Project Milestones
    2024-Q1 : Planning
            : Requirements gathering
    2024-Q2 : Development starts
            : MVP release
    2024-Q3 : Beta testing
    2024-Q4 : Production launch
```

## Multiple Events Per Period

```mermaid
timeline
    title Release History
    v1.0 : Initial release
         : Core features
         : Basic docs
    v2.0 : New UI
         : Performance improvements
    v3.0 : API overhaul
         : Plugin system
```

## Common Pattern: Company History

```mermaid
timeline
    title Company Milestones
    2020 : Founded
         : Seed funding
    2021 : Series A
         : 50 employees
    2022 : Product launch
         : 10K users
    2023 : Series B
         : International expansion
    2024 : 1M users
         : IPO preparation
```

## Common Pattern: Sprint Timeline

```mermaid
timeline
    title Sprint 5
    Week 1 : Sprint planning
           : Design review
    Week 2 : Core development
           : API integration
    Week 3 : Testing
           : Bug fixes
    Week 4 : Code review
           : Release
```
